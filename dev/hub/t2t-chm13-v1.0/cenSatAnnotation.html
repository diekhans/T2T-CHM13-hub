Description

Centromeric Satellite Annotation

Methods

Satellite array annotations are defined by intersecting information across alpha HOR annotation track, repeatmasker tracks, and human satellite annotation tracks. The start and end of each region is defined by centromere haplotypes (as published relative to GRCh38 coordinates).

Display Conventions and Configuration

Colors

  • Higher Order Repeats (hor): red
  • Monomeric (mon) repeats: orange
  • Human Satellites (hsat): blue
  • Centric Satellite Transition Regions (ct): grey
  • Beta Satellite (bsat): pink
  • Other centromeric satellites (censat): teal

Data access

Data is available on globus: team-satellite/t2t-chm13.20200727.cenSat_annotation.bed

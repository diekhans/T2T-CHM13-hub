Description

Merqury - k-mer multiplicity in 10XG reads

Methods

The first 23 bp of the 10X Genomics read pair 1 were trimmed to remove barcode bases, and 21-mers were collected with Meryl (read set). The multiplicity was annotated along the first base of the k-mer on the assembly with meryl-lookup -dump. Each beginning position (1-base) of the k-mer was annotated with the global copy number as a .wig file, and converted as bigWig format. As a result, the last 20 bases of an assembly are not included but follows the last 21 baseâ€™s copy number.

Display Conventions and Configuration

Maximum set to 100x as default. Adjust accordingly for repetitive regions.

Data access

Data is available on globus: team-curation/merqury/20200602.read_multiplicity.bigWig with a corresponding .tdf

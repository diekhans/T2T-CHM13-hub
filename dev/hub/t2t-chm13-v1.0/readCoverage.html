Description

Read coverage

Methods

Input reads: HiFi and ONT reads (guppy 3.6.0)
Mapper: Winnowmap v2.01 release (--MD -W repetitive_k15.txt -ax [map-pb | map-ont])
Filter: Primary read alignments extracted with samtools view -F0x104

Display Conventions and Configuration

Gray: Primary alignments. ONT reads were further filtered for alignment block lengths > 1 kb and > 85 % identity
Black: Marker assisted alignments

Data access

Data is available on globus: team-curation/tracks/v1/coverage/*.cov.wig

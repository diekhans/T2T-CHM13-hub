Description

Merqury - k-mer multiplicity in assembly

Methods

All 21-mers in the assembly were counted with meryl count k=21 and annotated along the first base of the k-mer on the assembly with meryl-lookup -dump. Each beginning position (1-base) of the k-mer was annotated with the global copy number as a .wig file, and converted as bigWig format. As a result, the last 20 bases of an assembly are not included but follows the last 21 baseâ€™s copy number.

Display Conventions and Configuration

Maximum set to 5x as default. Adjust accordingly for repetitive regions.

Data access

Data is available on globus: team-curation/merqury/20200602.asm_multiplicity.bigWig with a corresponding .tdf[b]

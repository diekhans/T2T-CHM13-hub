Description

Repeat Masker track constructed with the HMMER alignment engine and Dfam_3.1 database.

  • RepeatMasker version 4.1.0
  • Search Engine: HMMER [ 3.2.1 (June 2018) ]
  • RepeatMasker Database: HMM-Dfam_3.1

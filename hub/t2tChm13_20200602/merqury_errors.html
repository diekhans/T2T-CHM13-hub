Description

Merqury - Assembly-only (Errors)

Methods

The first 23 bp of the 10X Genomics read pair 1 were trimmed to remove barcode bases, and 21-mers were collected with Meryl (read set). Merqury was run on t2t-chm13.20200602.fasta with the read set, collecting assembly-only (error) k-mers. Each position containing an error k-mer was converted to a .bed file, with the accumulated number of k-mers per position collected in bigwig format. As a result, the maximum number of k-mers found per base will be 21 (all k-mers overlapping the base are errors).

Display Conventions and Configuration

Maximum set to 21x

Data access

Data is available on globus: team-curation/merqury/20200602.only.bigWig with corresponding 20200602.chr_only.tdf and 20200602.nonChr_only.tdf

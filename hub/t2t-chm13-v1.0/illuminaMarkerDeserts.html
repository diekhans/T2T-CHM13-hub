Description

Illumina Marker Assisted Mapping

Methods

Illumina Marker Assisted Mapping The data from this track was constructed using unique 21mers identified using merqury in Illumina PCRfree reads found when mapped to CHM13v1.0 (illumina_markers.bed). These data where also used to highlight illumina marker deserts where no 21mers were uniquely mapped.

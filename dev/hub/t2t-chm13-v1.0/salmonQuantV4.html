Description

Transcript-level expression estimates (CHM13).

Methods

TPM estimated using Salmon 1.3.0, CHM13 polyA RNA-seq (2 replicates combined), CHM13 v1.0 assembly, and CAT v4 + liftOff annotation.

Display Conventions and Configuration

Colored according to log2(TPM + 1e-6) in the following intervals:
  • Dark blue: (-19.97,-14.54]
  • Medium blue: (-14.54,-9.15]
  • Light blue: (-9.15,-3.76]
  • Gray: (-3.76,1.63]
  • Light red: (1.63,7.02]
  • Medium red: (7.02,12.41]
  • Red: (12.41,17.84]

The value of each feature corresponds to the raw TPM, rounded to the nearest integer and truncated at 1000 (BED convention).

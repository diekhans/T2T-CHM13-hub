Description

BLAT of GENCODE V33 to T2T CHM13 assembly

Methods

Methods are based on those used by the UCSC browser.

  1. Obtained GENCODE RNA sequence from http://gencodegenes.org (gencode.v33.lncRNA_transcripts.fa.gz gencode.v33.pc_transcripts.fa.gz) NCBI.
  2. Aligned to T2t CHM13 using blat -q=rna -fine -minScore=20 -stepSize=5 -noHead.
  3. Filter resulting alignment with pslCDnaFilter -minQSize=20 -ignoreIntrons -repsAsMatch -ignoreNs -bestOverlap -globalNearBest=0.001 -minId=0.92 -minCover=0.75

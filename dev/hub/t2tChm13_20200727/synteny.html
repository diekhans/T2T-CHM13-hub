Description

Synteny annotations relative to GRCh38 produce from the Cactus alignments.

Methods

These are build the Cactus HAL file with 25 kilobase and 1 megabase resolution. The 25kb track uses a maximum anchor distance of 25kb, while the 1 megabase track uses a 50kb block size.

Display Conventions and Configuration

These are alignment tracks using the PSL display conventions

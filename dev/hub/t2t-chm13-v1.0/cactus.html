Description

Cactus reference alignments GRCh38 and t2t-chm13-v1.0, with Chimp (GCF_002880755.1/panTro6) as an out-group.

Methods

Standard Cactus alignment portocol

Display Conventions and Configuration

Four tracks are shown:

  • Cactus GRCh38 - alignments of GRCh38 to t2t-chm13-v1.0
  • Cactus CHM13 - self-alignments of t2t-chm13-v1.0
  • Cactus GRCh38 simplified - alignments of GRCh38 to t2t-chm13-v1.0 using PSL display, which is faster to view.
  • Cactus GRCh38 chained - alignments of GRCh38 to t2t-chm13-v1.0 chained alignments

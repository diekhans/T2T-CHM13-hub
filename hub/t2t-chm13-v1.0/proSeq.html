Description

Precision Run-On (PRO) Sequencing STAR alignment

Methods

The PRO-seq experiment was done on CHM13 cells and sequenced for 75bp single-ended reads.

Reads were trimmed and filtered and then aligned to the T2T CHM13 v1.0 assembly (released on 20200922) using STAR/2.7.1a with the following parameters:

--outFilterMultimapNmax 2048

--outFilterMismatchNmax 999

--alignSJoverhangMin 8

--alignSJDBoverhangMin 1

--outFilterMismatchNoverReadLmax 0.04

--alignIntronMin 20

--alignIntronMax 1000000

--sjdbScore 1

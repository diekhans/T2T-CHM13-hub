Description

K* metric

Methods

A base-level measure of distance between the assembly and the raw data.
It is computed as the ratio between between the frequency of any given kmer in the assembly and its occurrence in the raw data.
At equilibrium K* = 0.
Input reads: Illumina PCRfree
Any significant deviation from the equilibrium is
A collapsed region if Kr >> Ka
An under-collapsed region if Ka >> Kr
Missing kmers are not highlighted (they equal 0)

Display Conventions and Configuration

The K* metric is centered on 0.

Data access

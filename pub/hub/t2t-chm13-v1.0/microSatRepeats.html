Description

Microsatellite repeats (%) in every 128 bp window, labeled with dimer notation in homopolymer compressed space. These tracks show the regions of CHM13 v1.0 with various simple-sequence patterns (GA/TC/GC/AT). The sequence is flagged even if one of the bases repeats (e.g. AAAATTTTAAATT is flagged as AT).

Methods

The track was generated using Seqrequester, a Meryl and Canu utility module. The tracks can be generated using seqrequester microsatellite.

Description

These track shows putative unique regions of T2T CHM13 relative to GRCh38 based on various methods This track shows regions that have no GRCh38 LASTZ Liftover alignment chains (GRCh38 LASTZ Liftover Alignment Chains track).

Methods

Base-wise negation of LAST alignment chains using featureBits:

  • GRCh38 LASTZ Liftover Alignment Chains - using only liftover changes
  • GRCh38 LASTZ Primary Liftover Alignment Chains - using only GRCh38 chromosomes, excluding chrY.

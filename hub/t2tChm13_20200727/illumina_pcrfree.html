Description

Illumina PCRfree mapping

Methods

Each pair of PCRfree Illumina read SRR1997411, SRR3189741 SRR3189742 SRR3189743 files in fastq format were mapped to t2t-chm13.20200727.fasta and mark duplicated read with bamsormadup using default parameter except SM=CHM13. All the mapped reads were then merged with bamtools version 2.3.0

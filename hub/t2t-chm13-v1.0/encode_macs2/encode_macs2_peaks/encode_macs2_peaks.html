Description

These tracks contain macs2 peak calls from Encode histone ChIP-seq experiments remapped using the T2T CHM13 v1.0 genome assembly.

Methods

Reads were aligned to the T2T CHM13 v1.0 assembly (released on 20200922) and filtered (see bam pileup track for details). Mapped reads were peak called using macs2 and the parameters "-f BAMPE -g 3.03e9". For the histone marks H3K9me3, H3K27me3, and H3K36me3 also used the parameter "--broad".

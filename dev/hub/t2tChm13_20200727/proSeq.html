Description

Precision Run-On (PRO) Sequencing alignment)

Methods

The PRO-seq experiment was done on CHM13 cells and sequenced for 75bp single-ended reads.

Reads were trimmed and filtered and then aligned to the T2T CHM13 assembly (20200727) using STAR/2.7.1a with the following parameters:

--outFilterMultimapNmax 100

--outFilterMismatchNmax 0

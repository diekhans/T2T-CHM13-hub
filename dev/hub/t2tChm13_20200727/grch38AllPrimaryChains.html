Description

This track is a filtering of the GRCh38 All Chains track to keep only GRCh38 sequences from the primary assembly that are in T2T CHM13. This excludes all patch, alternate loci, and chrY.

Methods

Filtered with the chainFilter command.

Display Conventions and Configuration

By default, the chains to chromosome-based assemblies are colored based on which chromosome they map to in the aligning organism. To turn off the coloring, check the "off" button next to: Color track based on chromosome.

To display only the chains of one chromosome in the aligning organism, enter the name of that chromosome (e.g. chr4) in box next to: Filter by chromosome.

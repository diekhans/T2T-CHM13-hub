Description

Transcript-level expression estimates (CHM13).

Methods

TPM estimated using Salmon 1.3.0, CHM13 polyA RNA-seq (2 replicates combined), CHM13 v1.0 assembly, and CAT v1 annotation.

Display Conventions and Configuration

Colored according to log2(TPM) in the following intervals:
  • Dark blue: (-23.3,-17.3]
  • Medium blue: (-17.3,-11.4]
  • Light blue: (-11.4,-5.45]
  • Gray: (-5.45,0.479]
  • Light red: (0.479,6.41]
  • Medium red: (6.41,12.3]
  • Red: (12.3,18.3]

The value of each feature corresponds to the raw TPM, rounded to the nearest integer and truncated at 1000 (BED convention).

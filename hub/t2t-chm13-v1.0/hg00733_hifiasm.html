Description

Alignments of HG00733 hifiasm assembly v0.11 contigs

Methods

A hifiasm assembly of HG00733 obtained from Heng Li and available at ftp://ftp.dfci.harvard.edu/pub/hli/hifiasm/HG00733-trio-v0.11/ was aligned to the T2T genome reference using nucmer v4.0.0beta2 with parameters "--maxmatch -l 100 -c 1000", as well as with minimap2 v2.17 with parameters "-ax asm5 -L". Nucmer alignments were converted to BAM format using the delta2sam.pl utility from SVanalyzer.

Description

These tracks contain bam pileups for Encode histone ChIP-seq experiments remapped using the T2T CHM13 v1.0 genome assembly.

Methods

All reads were downloaded from the Encode data portal. Reads were aligned to the T2T CHM13 v1.0 assembly (released on 20200922) using bowtie2 and the parameters "--no-discordant --no-mixed --very-sensitive --no-unal --omit-sec-seq --xeq".Mapped reads were filtered using samtools with the parameters "-F 1804 -f 2 -q 2". Mapped reads were then filtered for duplicates using picard and the parameters "VALIDATION_STRINGENCY=LENIENT ASSUME_SORT_ORDER=queryname REMOVE_DUPLICATES=true". Finally, mapped reads were filtered to only those containing at least one unique k-mer on either end.

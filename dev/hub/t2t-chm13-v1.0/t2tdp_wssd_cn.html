Description

This track represents copy number estimates for 20 T2T Diversity Panel genomes. Copy number is estimated over 500 bp windows of uniqely mappable sequence. Sequences for single copy number increments (0 - 9) and in 10 copy increments for 10+-120+. Exact copy can be found by clicking on the region of interest. \n\n

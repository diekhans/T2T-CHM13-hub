Description

This track illustrates repetitive 18mers of CHM13v1.

Methods

Kmers of 18bp in length were counted and mapped to the CHM13v1 genome assembly. If a kmer is identical in sequence to the adjacent kmer it is reported in the .bed file as a single unit.
